Начало работы

Вход в систему

Для входа в систему необходимо:

  1. Перейти на страницу входа
  2. Ввести ID пользователя
  3. Выбрать роль (администратор, модератор, пользователь)
  4. Нажать кнопку "Войти"
Тестовые пользователи:
  • Администратор: ID 40212
  • Модератор: ID 40213
  • Пользователь: ID 50000

Управление персонажами

Создание персонажа

Чтобы создать нового персонажа:

  1. Перейдите в раздел "Мои персонажи"
  2. Нажмите кнопку "Новый персонаж"
  3. Выберите игровую систему
  4. Укажите имя персонажа (опционально)
  5. Нажмите "Создать персонажа"
Редактирование персонажа

После создания персонажа вы можете:

  • Просматривать детальную информацию
  • Редактировать характеристики
  • Повышать уровень
  • Изменять оборудование
  • Просматривать историю изменений

Создание контента (Homebrew)

Homebrew - это пользовательский контент, который может включать:

  • Заклинания
  • Магические предметы
  • Классы персонажей
  • Расы
  • Способности и черты
Процесс создания:
  1. Перейдите в раздел "Мой контент"
  2. Нажмите "Создать контент"
  3. Выберите тип контента
  4. Укажите игровую систему
  5. Заполните название и описание
  6. Добавьте детальное содержание в JSON формате
  7. Сохраните как черновик или отправьте на модерацию

Модерация контента

Для модераторов и администраторов доступны функции проверки контента:

  • Очередь модерации: Список записей, ожидающих проверки
  • Проверка записи: Детальный просмотр контента
  • Решение: Одобрить или отклонить контент
  • Комментарии: Указание причины отклонения

Статистика и аналитика

Доступные статистические данные:

  • Статистика пользователей: Активность, созданный контент
  • Статистика сессий: Длительность, действия пользователей
  • Статистика систем: Использование игровых систем
  • Модерационная статистика: Эффективность модераторов

API документация
