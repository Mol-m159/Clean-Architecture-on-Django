НРИ Аналитика

Система анализа пользовательской активности для платформы настольных ролевых игр.

Основные функции:
  • Создание и управление персонажами
  • Разработка пользовательского контента (homebrew)
  • Модерация пользовательского контента
  • Статистика пользовательской активности
  • Аналитика использования игровых систем
Роли пользователей:
Администратор

Полный доступ ко всем функциям системы, просмотр статистики, управление пользователями.

Модератор

Проверка пользовательского контента, одобрение или отклонение записей.

Пользователь

Создание персонажей, разработка контента, просмотр материалов.

Технологии:
  • Backend: Django, Django REST Framework
  • Frontend: Bootstrap 5, Chart.js
  • База данных: Microsoft SQL Server
  • API: RESTful API
